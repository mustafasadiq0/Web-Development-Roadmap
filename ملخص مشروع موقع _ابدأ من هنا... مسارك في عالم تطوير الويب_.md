## الأقسام الرئيسية

### 1. القسم الرئيسي (Hero Section)
- عنوان جذاب: "ابدأ من هنا... مسارك في عالم تطوير الويب"
- إحصائيات مهمة: نمو الطلب (+50%)، متوسط الراتب (15,000 ريال)، مدة التعلم (6-12 شهر)
- أزرار دعوة للعمل: "ابدأ رحلتك الآن" و "اكتشف المسارات"

### 2. قسم "ما هو تطوير الويب؟"
- تعريف شامل لتطوير الويب
- أربع بطاقات توضيحية:
  - مواقع الويب
  - تطبيقات الويب
  - تجربة المستخدم
  - فرص العمل

### 3. قسم "اختر المسار المناسب لك"
ثلاث بطاقات تفصيلية للمسارات:

#### أ) تطوير الواجهة الأمامية (Frontend)
- المستوى: مبتدئ
- التقنيات: HTML, CSS, JavaScript, React, Vue.js, Angular
- مدة التعلم: 6-9 أشهر
- الراتب المتوقع: 8-15 ألف ريال
- مناسب لمن يحب التصميم والتفاعل المرئي

#### ب) تطوير الواجهة الخلفية (Backend)
- المستوى: متوسط
- التقنيات: Python, Node.js, PHP, MySQL, PostgreSQL, MongoDB
- مدة التعلم: 8-12 شهر
- الراتب المتوقع: 10-18 ألف ريال
- مناسب لمن يحب المنطق والحلول التقنية

#### ج) تطوير الويب الشامل (Fullstack)
- المستوى: متقدم
- التقنيات: جميع تقنيات Frontend + Backend, MEAN, MERN, LAMP
- مدة التعلم: 12-18 شهر
- الراتب المتوقع: 12-25 ألف ريال
- مناسب لمن يريد فهم شامل للتطوير

### 4. مكتبة الموارد التعليمية
قسم مقسم إلى جزأين:

#### الدورات العربية:
- **أكاديمية حسوب**: دورات تطوير الواجهات الأمامية، PHP، JavaScript المتقدمة
- **منصة برمج**: معسكر Full Stack Developer، دورات React و Node.js
- **أكاديمية طويق**: معسكرات Python و Java
- **قناة الزيرو (Elzero)**: مسار Frontend Developer الكامل

#### الدورات الإنجليزية المجانية:
- **freeCodeCamp**: شهادات في Responsive Web Design، JavaScript
- **The Odin Project**: مسارات Full Stack JavaScript و Ruby on Rails
- **CS50**: دورة هارفارد في تطوير الويب
- **منصات التطوير التفاعلية**: CodePen، Replit، GitHub Codespaces

### 5. مكتبة الأدوات

- **التحكم في الإصدارات**: Git & GitHub، GitLab، Bitbucket

## الميزات التقنية

### التصميم والواجهة:
- **اللغة**: العربية مع دعم كامل للكتابة من اليمين إلى اليسار (RTL)
- **الخط**: Cairo - خط عربي حديث وأنيق
- **نظام الألوان**: ألوان هادئة ومريحة (أزرق وأخضر)
- **التصميم المتجاوب**: يعمل بشكل مثالي على جميع الأجهزة
- **التفاعلات**: حركات وتأثيرات بصرية جذابة

### التقنيات المستخدمة:
- **React**: إطار العمل الرئيسي
- **Tailwind CSS**: للتصميم والتنسيق
- **Lucide Icons**: للأيقونات
- **shadcn/ui**: لمكونات الواجهة
- **Vite**: لبناء المشروع

### الأداء والجودة:
- **سرعة التحميل**: محسن للأداء السريع
- **SEO**: محسن لمحركات البحث
- **إمكانية الوصول**: متوافق مع معايير الوصول
- **التوافق**: يعمل على جميع المتصفحات الحديثة
